What Is a For Loop?

A for loop allows repeatedly executing a block of code while a condition is true.

It has 3 parts: initialization, condition, and increment/decrement.

Syntax

for (initialization; condition; increment/decrement) {
  // code to be executed
}

Example

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

This prints numbers 1 to 5 in the console.
